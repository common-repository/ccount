This plugin adds a widget showing the commenters and the number of their comments.

== Description ==

This plugin adds a new widget to the dashboard. It shows a list of all commenters and the number of their comments. This list is ordered by the number of comments they made. The e-mail address is used to distinguish the commenters.

== Installation ==

1. Upload all files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enjoy the plugin on the Dashboard
